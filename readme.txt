== Description ==

Convert your WordPress post to markdown with a click of a button!
This plugin helps you cross-post to Dev.to by generating the markdown for a Dev.to post.
This plugin adds a button to the frontend for admin users. When you click the button, the post is converted to markdown via a wp-json API. 

Requires php-dom on server:

sudo apt-get install php-dom

== Installation ==

Requires php-dom on server:

sudo apt-get install php-dom
